#ГЛАВНАЯ

nano /etc/apt/sources.list - перед скачиванием пакетов закомментируйте первую строку

nano /etc/resolf.conf - также для скачивания пакета необходимо установить DNS-сервер, ставим nameserver 8.8.8.8 (остальные строки закомментируйте).

❗ После настройки DNS-сервера на HQ-SRV на остальных машинах нужно поставить nameserver <ip-адрес HQ-SRV> ❗

❗❗КРОМЕ ISP, НА iSP RESOLV.CONF НЕ МЕНЯЕМ❗❗

apt update - обновление пакетов. Некоторые пакеты не будут скачиваться без обновления (например, bind9). Если вы видите, что система не устанавливает пакет и просит его обновить, сделайте это.
